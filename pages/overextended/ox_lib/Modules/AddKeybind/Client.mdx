# Client

Registra atalhos de teclado e simplifica as interações dos atalhos.

## CKeybind Class

Uma tabela que representa um atalho de teclado com as seguintes propriedades.

- name: `string`
- description: `string`
- teclaAtual: `string`
  - Tecla que o usuário atual definiu para este atalho de teclado
- disabled: `boolean`
  - Se o atalho de teclado está atualmente desativado
- hash: `number`
  - Hash interno do atalho de teclado que é usado para referenciá-lo dentro do próprio jogo
- defaultKey?: `string`
  - Tecla padrão para definir o atalho de teclado para novos jogadores
  - NOTA: Alterar isso não mudará a tecla para jogadores existentes
- defaultMapper?: `string`
  - Veja [IDs de Mapeador de Entrada](https://docs.fivem.net/docs/game-references/input-mapper-parameter-ids/) para mais informações
- secondaryKey?: `string`
  - Uma tecla secundária opcional.
- secondaryMapper?: `string`
  - Um mapeador opcional para a tecla secundária, caso contrário, usando o mapeador padrão.
- disable: `function(self: CKeybind, disable: boolean)`
  - Função interna para habilitar / desabilitar um atalho de teclado
- onPressed?: `function(self: CKeybind)`
  - Função definida pelo usuário acionada ao pressionar o atalho de teclado
- onReleased?: `function(self: CKeybind)`
  - Função definida pelo usuário acionada ao soltar o atalho de teclado

## lib.addKeybind

```lua
lib.addKeybind(data)
```

- data: `table`
  - name: `string`
  - description: `string`
  - defaultKey?: `string`
    - Default: `None`
  - defaultMapper?: `string`
    - Default: `keyboard`
  - secondaryKey?: `string`
  - secondaryMapper?: `string`
  - disabled?: `boolean`
    - Se o atalho de teclado deve estar desativado por padrão
  - onPressed?: `function(self: CKeybind)`
    - Function triggered on keybind press
  - onReleased?: `function(self: CKeybind)`
    - Function triggered on keybind release

```lua
local keybind = lib.addKeybind({
    nome = 'respeitos',
    descrição = 'pressione F para prestar respeitos',
    defaultKey = 'F',
    onPressed = function(self)
      print(('pressionado %s (%s)'):format(self.teclaAtual, self.nome))
    end,
    onReleased = function(self)
      print(('solto %s (%s)'):format(self.teclaAtual, self.nome))
    end
})
```

### Habilitar / Desabilitar Atalhos de Teclado

Atalhos de teclado podem ser habilitados / desabilitados usando o método disable.

```lua
keybind:disable(true) -- desativa o atalho de teclado
keybind:disable(false) -- ativa o atalho de teclado
```
