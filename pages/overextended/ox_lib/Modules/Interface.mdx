# Interface

Se você deseja alterar a cor primaria da interface para combinar melhor com o tema do seu servidor, você pode fazer isso facilmente através dos convars disponíveis.

Não é necessário compilar a interface, basta reiniciar o recurso.

Convars:

```yaml
setr ox:primaryColor blue
setr ox:primaryShade 8
```

A alteração da cor primária altera a cor de elementos como a barra/círculo de progresso, a verificação de competências, o botão central do menu radial e o foco, os botões de confirmação de diálogo, o foco do campo de entrada e muito mais..

Pode encontrar a lista completa de cores e tonalidades predefinidas aqui:

https://v6.mantine.dev/theming/colors/#default-colors

Se quiser criar a sua própria palete, sugiro que siga o guia nessa página.
Não se esqueça de que, para o fazer, terá de descarregar o código-fonte e construir a IU.
