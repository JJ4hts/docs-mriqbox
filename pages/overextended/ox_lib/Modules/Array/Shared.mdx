# Shared

## lib.array

Uma tabela usada como protótipo para a classe Array. Funções podem ser usadas como métodos de classe ou funções padrão.

### lib.array:new

Constrói uma instância de Array contendo os elementos fornecidos.

```lua
lib.array:new(...)
```

- ...: `any`

**Returns:**

- arr: `Array`

### lib.isArray

Determina se a tabela fornecida é uma instância de Array ou uma tabela semelhante a um array.

```lua
lib.array.isArray(tbl)
```

- tbl: `table`

**Returns:**

- isArray: `boolean`

### lib.array.merge

Combina os elementos de dois arrays em um novo array.

- a: `Array`
- b: `Array`

```lua
lib.array.merge(a, b)
```

**Returns:**

- Array

### lib.array.every

Verifica se todos os elementos de um array passam na função de teste fornecida.

- arr: `Array`
- testFn: `function`

```lua
lib.array.every(arr, testFn)
```

**Returns:**

- success: `boolean`

### lib.array.filter

Cria um novo array contendo os elementos de um array que passam na função de teste fornecida.

- arr: `Array`
- testFn: `function`

```lua
lib.array.filter(arr, testFn)
```

**Returns:**

- arr: `Array`

### lib.array.find

Retorna o primeiro elemento de um array que passa na função de teste fornecida.

- arr: `Array`
- testFn: `function`
- reverse?: `boolean`
  - Itera sobre o array em ordem reversa.

```lua
lib.array.find(arr, function(element) end, reverse)
```

**Returns:**

- element: `unknown`

### lib.array.findIndex

Retorna o índice do primeiro elemento de um array que passa na função de teste fornecida.

- arr: `Array`
- testFn: `function`
- reverse?: `boolean`
  - Itera sobre o array em ordem reversa.

```lua
lib.array.findIndex(arr, function(element) end, reverse)
```

**Returns:**

- index: `number`

### lib.array.indexOf

Retorna o índice do primeiro elemento de um array que corresponde ao valor fornecido.

- arr: `Array`
- value: `any`
- reverse?: `boolean`
  - Itera sobre o array em ordem reversa.

```lua
lib.array.indexOf(arr, value, reverse)
```

**Returns:**

- index: `number`

### lib.array.forEach

Executa a função fornecida para cada elemento de um array.

- arr: `Array`
- cb: `function`

```lua
lib.array.forEach(arr, function(element) end, reverse)
```

**Returns:**

- index: `number`

### lib.array.join

Concatena todos os elementos de um array em uma string, separados por vírgulas ou pelo separador especificado.

- arr: `Array`
- seperator?: `string`

```lua
lib.array.join(arr, seperator)
```

**Returns:**

- str: `string`

### lib.array.pop

Remove o último elemento de um array e retorna o valor.

- arr: `Array`

```lua
lib.array.pop(arr)
```

**Returns:**

- elemento: `desconhecido`

### lib.array.push

Adiciona os elementos fornecidos ao final de um array e retorna o novo comprimento do array.

- arr: `Array`
- ...: `any`

```lua
lib.array.push(arr, ...)
```

**Returns:**

- length: `number`

### lib.array.shift

Remove o primeiro elemento de um array e retorna o valor.

- arr: `Array`

```lua
lib.array.shift(arr)
```

**Returns:**

- elemento: `desconhecido`

### lib.array.reduce

A função "reducer" é aplicada a cada elemento de um array, com o resultado anterior servindo como acumulador.  
Se um valor inicial for fornecido, ele é usado como o acumulador para o primeiro índice; caso contrário, a iteração começa no segundo índice, com o primeiro índice como o acumulador.

- arr: `Array`
- reducer: `function`
- initialValue?: `any`

```lua
lib.array.reduce(arr, function(accumulator, element, index) end)
```

**Returns:**

- acumulador: `desconhecido`
